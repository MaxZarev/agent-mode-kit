# Стиль постов канала

Опиши здесь голос и стиль своего канала — агент будет следовать ему при
написании и редактуре постов. Чем конкретнее, тем лучше. Примеры пунктов:

- Тон: партнёрский / экспертный / ироничный / формальный…
- Конкретика: цифры вместо абстракций («30 минут вместо 6 часов»)
- Фирменные приёмы: контрасты «Раньше… Теперь…», риторический вопрос отдельной строкой
- Чего избегать: пафос, мотивационные клише, КАПС без причины, канцелярит
- Эмодзи: сколько на пост, какие, анимированные или обычные
- Чем заканчивается пост: вопрос, CTA, вывод

Если файл не заполнен — агент уточняет стиль у пользователя перед написанием поста.
